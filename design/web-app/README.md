# Milk Delivery Service — Responsive Web UI (React)

This folder contains a production-ready UI/UX specification for a premium, mobile‑first web application implemented in React and Tailwind. It complements the existing Django REST backend without changing backend logic.

- Design tokens: tokens.json (colors, typography, spacing, radius, shadows, motion, breakpoints)
- Component library: components.md (atoms, molecules, organisms with props and variants)
- Screens: screens.md (page layouts, sections, responsive rules, data placeholders)
- Interactions: interactions.md (transitions, micro‑interactions, motion curves)
- Imagery: imagery.md (hero/product sources, optimization and art direction)

Use an 8px grid, auto layout principles, and component variants for states. Palette blends white/blue with fresh green accents and subtle glassmorphism for a premium dairy brand feel.
