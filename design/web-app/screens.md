# Screen Specifications — Milk Delivery Service

## 1. Landing Page (Public)
- **Hero**: Full-width split layout. Left: High-impact headline ("Pure Daily Freshness") + CTAs. Right: High-res milk splash or product hero image.
- **Features**: 3-column grid with custom icons and soft glass cards.
- **How it Works**: Vertical timeline on mobile, horizontal steps on desktop.
- **Plans**: 3-card layout (Standard, Premium, Family) with "Popular" badge on Premium.

## 2. User Dashboard (Protected)
- **Greeting**: "Good morning, [User]! 🥛" with a weather-based delivery status message.
- **Delivery Card**: Large surface with current delivery progress (e.g., "Out for delivery").
- **Grid Layout**: 
  - Top: Wallet & Subscriptions summary.
  - Middle: Quick Actions (horizontal scroll on mobile).
  - Bottom: Upcoming 7-day timeline.

## 3. Product Listing
- **Grid**: Responsive 1 col (mobile) -> 3 col (tablet) -> 4 col (desktop).
- **Sticky Header**: Breadcrumbs + Result count + Mobile filter trigger.
- **Empty State**: Custom illustration for "No products found".

## 4. Subscription Management
- **Main View**: Interactive calendar with status indicators (Green: Active, Grey: Paused, Red: Skip).
- **Side Panel**: Details of the selected date's delivery with an "Edit for this day only" option.
- **Toggles**: Global "Pause All" with date picker.

## 5. Cart & Checkout
- **Split View (Desktop)**: Left: Items list + Address selector. Right: Sticky Order Summary card.
- **Payment**: Clean radio buttons for Wallet, Card, UPI, or Cash on Delivery.
- **Success Page**: Celebration animation (Lottie) + "Track Order" CTA.

## 6. Profile & History
- **Tabs**: Personal Info, Saved Addresses, Billing, Order History.
- **History**: Collapsible rows with "Reorder" button.
- **Address**: Card grid with "Default" tag.
