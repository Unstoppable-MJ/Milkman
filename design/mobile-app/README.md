# Milk Delivery App — Mobile UI (Figma-Ready)

This folder contains a production-ready design specification for an Android mobile app, structured for quick import into Figma as components, frames, and tokens.

- Design tokens: tokens.json (colors, type, spacing, radius, shadows, motion)
- Component library: components.md (names, variants, auto layout, constraints)
- Screens: screens.md (layouts, sections, states, frame sizes)
- Imagery: imagery.md (image/illustration sources and guidelines)
- Interactions: interactions.md (transitions, micro-animations, haptics)

Use 8dp grid, auto layout everywhere, component variants for states, and Material Symbols Rounded for icons. Primary palette blends white/blue with fresh green accents for a premium dairy feel.
